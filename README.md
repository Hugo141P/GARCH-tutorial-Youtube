# GARCH tutorial series 
For questions, please leave a comment under the Youtube video.

First create a venv: (linux)
```python3 -m venv /root/garch-tutorial-youtube/venv```
Or the corresponding commands that work on your system.
Install dependencies:
```pip install -r requirements.txt```

## Tutorial 1
Basic implementation of a GARCH(1, 1) model with a normal distribution for the errors.
## Tutorial 2
Added the t distribution.
## Tutorial 3
Added the GED and EGB2 distribution, along with a framework to implement any distribution.
## Tutorial 4
Added Value at Risk calculation by implementing cdf and inverse CDF functions.
## Tutorial 5
Added Expected Shortfall.
